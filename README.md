# HYDRA Burn

A decentralized token burn interface for HYDR tokens on the Radix Network.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Radix Dapp Toolkit

## Getting Started

```sh
# Clone the repository
git clone https://github.com/HYDRAXRD/hydraburn

# Navigate to the project directory
cd hydraburn

# Install dependencies
npm i

# Start the development server
npm run dev
